## 🌟 What is BookSphere?

BookSphere is a **super cool platform** where book lovers can:
- 📖 **Exchange books** with people nearby
- 💰 **Earn points** for sharing
- 🏆 **Level up** from Bronze to Gold Reader
- 🌳 **Help save trees** by reusing books

> 🎉 Think of it as **Instagram + OLX for books**, but way more fun!
